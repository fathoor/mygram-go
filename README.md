## API Endpoints

### Users

| Method | Endpoint | Description |
| :---: | :---: | :---: |
| **POST** | `/users` | Create new user |
| **GET** | `/users` | Get all users |
| **PUT** | `/users/:userId` | Update user by `userId` |
| **DELETE** | `/users/:userId` | Delete user by `userId` |
